Login to your ubuntu server and execute the following steps below 
and use "Git Bash" to login remotely to the server.


------------------------------------------
Step-01: Clone Deployment Repository 
------------------------------------------

"REPO-BRANCH" - Repository Branch Name. Provided by RAMESES.

"LGU-TYPE"    - LGU Type can be of the following: 

                * city 
                * province 
                * municipality  

## Go to User Home Directory
## 

$ cd 


## Clone the deployment repository
## 

$ git clone -b {province-municipality}  https://github.com/ramesesinc/etracs-{LGU-TYPE}-deployments.git  docker 

    //github user:etracslgurepo

## Go to docker SH files
## 

$ cd docker/bin


## Download all docker images needed for this deployment
## 

$ sh update-docker-images.sh



## List all downloaded docker images 
## 

$ docker images
