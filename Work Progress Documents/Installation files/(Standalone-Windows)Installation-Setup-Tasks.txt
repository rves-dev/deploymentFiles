===============================
Steps To Execute (Server Side) 
===============================

01. Install MySQL Server

02. Install Java 1.8

03. Clone Repository (Municipality)

    3.01 Create a directory "SERVERS" in drive C:

    3.02 Go to folder "SERVERS" and open a Git Base console 
       and start cloning the repo:

    URL   : https://github.com/ramesesinc/etracs255-server-{LGU-Type}.git 
    Branch: depteam


04. Configure Repository

    04.01  Go to folder "etracs255-server-{LGU-Type}"

    04.02  Unpack the custom template

           $  tar -xvf custom-template.tar.gz


    04.03  Open the file "bin/env.conf" 

    04.04  Change the value of the following settings:    

           lgu_name=MUNICIPALITY OF TESTER       

           app_server_ip=192.168.0.26       

           dbserver_host=192.168.0.1
           dbserver_user=root
           dbserver_pass=P@ssw0rd#       

           db_etracs=etracs255_tester


    04.05  Save and close the text editor.


05. Create the etracs main database 

    05.01  Go to folder "etracs255-server-{LGU-Type}/_res/_initial" 
           and open a Windows Command Prompt console

    05.02  Create the etracs main database 

    mysql -u root -p -e "CREATE DATABASE etracs255_tester CHARACTER SET utf8"


06. Restore Initial Databases

    mysql -u root -p -f  <  06-eor.sql

    mysql -u root -p -f -D etracs255_tester  <  01-etracs255-schema.sql
    
    mysql -u root -p -f -D etracs255_tester  <  02-etracs255-data.sql
    
    mysql -u root -p -f  <  03-etracs_notification.sql

    mysql -u root -p -f  <  04-etracs_image.sql
    
    mysql -u root -p -f  <  05-queue.sql

    
07. Open the file "ext/set-{LGU-Type}-data.sql"

08. Modify the data parameters and save the changes.
    Close the text editor to proceed to the next step.

09. Patch the extended script to the etracs main database.

    mysql -u root -p -f -D etracs255_tester < ext/set-{LGU-Type}-data.sql
    

10. Go to folder "etracs255-server-{LGU-Type}/bin" 
    and create a desktop shortcut of the following files: 

    ** run.bat
    ** shutdown.bat

    NOTES:
    - Change the shortcut label of "run.bat" to "Start ETRACS"
    - Change the shortcut label of "shutdown.bat" to "Stop ETRACS"


11. Go to your Desktop, double click the file "Start ETRACS" 
    to boot start your ETRACS application server.

12. Proceed to Client Installation.
