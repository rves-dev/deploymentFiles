03. Shutdown the old ETRACS app server 

    03.01  Using the Windows File Manager,

    03.02  Run the file "shutdown.bat" to stop the app server

    03.03  Close the shutdown console window

05. Open Navicat and login to the database server


06. Open a "New Query" window and do the following:

    06.01  Press CTRL + O to show the open dialog
    
    06.02  Go to folder "{etracs-server-dir}/_res/_upgrade"
    
    06.03  Open files sequentially starting from 01  
    
    06.04  Select the etracs main database first 

    06.05  Press CTRL + R for each file 
           to execute and apply the updates.

    06.06  Repeat Step-06.01 until all files are executed.

    
    Proceed to the next step if all scripts 
    are successfully executed.



07. Run ETRACS app server 


    07.01  Using the Windows File Manager, 
           locate the folder "{etracs-server-dir}/bin"


    07.02  Run the file "run.bat" to start the app server

09. Inform everyone that they can now 
    open the ETRACS client and login
