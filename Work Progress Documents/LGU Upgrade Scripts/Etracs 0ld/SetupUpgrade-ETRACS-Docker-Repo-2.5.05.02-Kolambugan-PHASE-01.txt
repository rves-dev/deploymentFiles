01. Login to the server via Git Bash

    $  cd 

    $  git clone -b ldn-kolambugan https://github.com/ramesesinc/etracs-municipality-deployments.git docker

    Proceed to Step-03 if pull is successful 
    otherwise fix the errors on Step-02


03. Download the latest docker images

    $  cd  ~/docker/bin

    $  sh  update-docker-images.sh


    Proceed to Step-04 if update is successful 
    otherwise fix the errors on Step-03


04. Extract the custom template

    $  cd  ~/docker

    $  tar -xvf custom-template.tar.gz
    

05. Open the file "env.conf" using the "nano" editor

    $  cd  ~/docker

    $  nano _custom/env.conf


06. Change the value of the following settings:

    lgu_name=MUNICIPALITY OF KOLAMBUGAN

    app_server_ip=

    dbserver_host=
    dbserver_user=
    dbserver_pass=

    db_etracs=etracs255_kolambugan



    NOTE: To save, press CTRL + X, then press Y, then press ENTER
          To cancel, press CTRL + X, then press N


07. End of Phase-1 Setup

    The setup will resume after the server is tranferred to Kolambugan 
    since the setup is done in the Province.
